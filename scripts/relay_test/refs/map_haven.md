# Valorant — Haven Map Reference

**Status:** Current as of June 2026  
**Pool status:** Active competitive map  
**Defining feature:** Only map in Valorant with **three spike sites** (A, B, C)  
**Setting:** Bhutan monastery / walled monastery town aesthetic  
**Theme:** Stone walls, wooden gates, garden courtyards, pagoda towers

---

## Map Overview

Haven has three bomb sites arranged across a wide horizontal layout. Attackers enter from a central spawn and split toward A (left), B (centre), or C (right). Defenders spawn centrally between all three sites, giving faster rotations but more area to cover. Mid (Garage / Courtyard) is the strategic linchpin — the team that controls it controls rotation timing for both B and C simultaneously.

**Layout sketch (left → right):**
```
[Attacker Spawn]
      |
  A Lobby ──── A Long ──── A Site ──── A Link ────┐
      |                                            |
  A Short (Sewer) ──────────────────────────── A Link
                                                   |
                              Mid Courtyard ───── CT / Defender Spawn
                                   |
                            Mid Doors (Double Doors)
                                   |
                     Garage ──── Nest (Window above Garage)
                       |           |
                   C Short     C Window ──── B Site
                       |
                   C Long ──── C Link ──── C Site ──── C Link ──── CT
```

## Strategic Notes for Relay Corpus

- **Garage is the map's pivot** — winning Garage simultaneously pressures B (via Window) and C (via Short); losing it forces site defenders to react blind.
- **A Heaven control** = free information and angle on both A Long and Sewer; almost mandatory in post-plant.
- **Hell** is the single hardest post-plant spot to clear; a defender holding Hell forces attackers to commit utility before they can defuse.
- **Three sites** means defenders are always one short; mid Garage control forces the third defender to over-extend to cover two sites.
- **C Long** is sniper-alley; Operator duels here are common; Breach/Skye/KAY/O flash utility into Long is standard execute opener.
- **B Site** is the "safe" execute — with A Link + C Link smoked and a Window pop-flash, it is the cleanest three-man push on the map.
- **Double Doors (mid doors)** are breakable/armoured; listening for the door-break audio is a key info cue.
- **Nest** above Garage is a popular defender perch covering Window; attackers should smoke/flash Nest on any Garage push.
