# Valorant — Breeze Map Reference

**Status:** Current as of June 2026  
**Pool status:** Active competitive map (returned Patch 12.00 / 2026 Season Act I with major rework)  
**Defining feature:** Largest map in Valorant; wide-open, long sightlines; heavy Operator/rifle map; now narrowed by rework  
**Setting:** Tropical paradise in the Bermuda Triangle — beach ruins, ancient stone structures, cannons, arches, market stalls  
**Theme:** Caribbean / Atlantic colonial-era fort ruins; sandy beaches, azure water visible in background

> **REWORK NOTE (Patch 12.00, 2026 Season Act I):** Breeze received the most extensive rework in Valorant's history after being removed from the pool in June 2024. Key structural changes: A Hall / A Link area removed or heavily modified; mid narrowed into corridor-style lanes; B site tightened; A Ropes removed from attacker hall; Mid Nest jump-up now requires mobility abilities; B attacker-spawn-side window path removed. Callout names for legacy zones are preserved below with rework notes where relevant.

---

## Map Overview

Breeze is a three-lane map with two bomb sites (A and B) connected by a large mid section. Attackers enter from the east; defenders spawn centrally on the west side. Mid is the strategic heart — it is larger than both sites combined and provides access to both A and B simultaneously. The rework (2026) tightened mid from an open plain into defined corridors, making it more tactical and less pure-sniper.

**Layout sketch (simplified, attacker-left to defender-right):**
```
[Attacker Spawn]
      |
  A Lobby / Cave ──── A Shop ──── A Main (east) ──── A Site ──── Defender Spawn (via Arches/Halls)
       |                                                |
      Rope ──── A Hall ──── Switch ──── Metal Doors ──┘
       |
  Mid Bottom (Cannon side) ──── Mid Pillar ──── Mid Top (Nest) ──── Mid Wood Doors ──── A Site
       |
  B Elbow ──── B Tunnel ──── B Site ──── Arches ──── Defender Spawn
       |
  B Main / B Window / Snake ──── B Site
```

## Rework Summary (Patch 12.00 — 2026 Season Act I)

### Removed / Eliminated
- A Link (wooden ledge connector) — completely gone
- Attacker-side A Rope in Hall — removed
- Snake (attacker spawn-to-B-Window side path) — removed entirely; only one non-mid B path remains
- Old A Stairs (mid access from stairs replaced with a small mid passage)
- Nest jump-up (now ability-gated: Jett updraft / Omen TP / Raze satchel required)

### Changed / Modified
- A Hall — heavily reworked; new enclosed corridor design connecting to mid more logically
- Mid — converted from open expanse to corridor-style lanes with cover; central wall expanded
- Mid Pillar — now connected to larger wall (not isolated)
- Mid Nest — smaller space; box repositioned; requires ability to access jump-up
- B Wall — extended toward B Main; prevents multi-angle coverage from single position
- B Cubby — significantly expanded due to extended wall; better post-plant cover
- B Window — widened
- A Pyramids — made taller for better spike cover
- B Back walls — raised to reduce angle count attackers must clear
- A Shop entrance — widened
- Mid Wood Doors — enclosed with utility windows (info-only gaps)
- B Elbow — enclosed with utility windows; wall no longer climbable
- A Main — new impenetrable wall sections tighten the entry

### Philosophy
Tighten open spaces, reduce angle complexity, improve defender rotation times, increase controller-agent viability, reduce pure-sniper dominance.

## Agent Synergies (Breeze-Specific)

### Strong Picks (current reworked Breeze)
- **Viper** — wall + snake essential for mid corridor control; smokes for long sightlines
- **Jett** — dash into mid Nest (now requires ability to access); Op mobility in long corridors
- **Sova** — recon darts into large open areas; scan through mid + sites
- **Chamber** — Operator anchoring in long mains and mid; teleport to escape
- **Fade** — info-gathering through wide corridors and around corners; haunt into mid
- **Deadlock** — strong post-plant and tight corridor utility (top agent on current Breeze per meta stats)
- **Vyse** — strong sentinel for the enclosed reworked spaces (top agent on current Breeze per meta stats)

### Difficult Picks
- **Phoenix** — limited utility range in large spaces
- **Reyna** — limited info/utility for wide-open fights
- **Harbor** — less effective; Viper dominates controller slot
