# Valorant — Lotus Map Reference

**Status:** Current as of June 2026  
**Pool status:** Active competitive map (Season 2026 Act 3 — Ascent, Breeze, Fracture, Haven, Lotus, Pearl, Split)  
**Defining features:** Three spike sites; two **rotating doors** (A-side and C-side); one **destructible wall** (A Link → B); **silent drop** (A Top → A Drop)  
**Setting:** Ancient Indian temple complex in a jungle river delta  
**Theme:** Carved stonework, lush vegetation, waterfalls, moss-covered ruins, lotus-flower motifs

---

## Map Overview

Lotus is a three-site map — the second such map after Haven. Attackers spawn near a lake and split into three lanes: A Root/Rubble lane (left), B Pillars lane (centre-left), and C Lobby lane (right). Defenders spawn centrally at CT/Waterfall and have fast rotations inward, but the map's sheer size makes cross-site rotations slow. Two rotating doors (A Door, C Door) create dynamic rotation shortcuts audible map-wide. A single destructible wall (B Door) permanently opens A Link into B Site when broken. B site is the smallest and most straightforward; A and C are larger and more complex.

**Layout sketch (top-to-bottom, attacker perspective):**
```
[Attacker Spawn / T Spawn]
         |
   A Lobby ─── A Root / A Rubble ──── A Main ──────────────────────────── A Top (Heaven)
         |              |                  |                                      |
      A Root         A Door (Rotating)  A Link ─── (Destructible Wall) ─── B Site ─ B Upper
         |              |                  |                                      |
         └── A Tree ────┘              B Main / B Pillars               CT Spawn / Defender Spawn
                                          |                                      |
                                      C Lobby ─── C Mound ─── C Door (Rotating) ─── B Main
                                          |                         |
                                      C Main ─── C Bend ─── C Site ─── C Waterfall ─── CT
                                                    |              |
                                                C Gravel ──── C Hall ──── C Link ──── CT Spawn
```

## Rotating Doors — Mechanics Reference

| Feature | Detail |
|---|---|
| **A Door location** | Between A Main / A Rubble and A Tree |
| **C Door location** | Between C Mound and B Main |
| **Activation** | Toggle button / switchboard panel on both sides of each door |
| **Rotation time** | ~10 seconds to complete 180° rotation |
| **Audio cue** | Loud, distinctive sound audible across the entire map; highlighted on minimap with a door icon |
| **Utility interaction** | Any agent utility placed in the door's sweep path is **instantly destroyed** |
| **Tactical note** | Opening either door telegraphs your rotation intent; defenders and attackers both listen for the audio; can be used as a distraction/fake rotation |
| **Minimap indicator** | A door icon appears on the minimap when a rotating door is in motion |

---

## Destructible Wall — Mechanics Reference

| Feature | Detail |
|---|---|
| **Location** | A Link corridor between A Main and B Site |
| **HP** | 400 HP |
| **Health indicator** | Colour-coded: Blue (healthy) → Yellow (damaged) → Red (near-broken) |
| **Audio cue** | Very loud noise when broken; heard across the map; major intel cue for defenders |
| **Permanence** | Once broken, the passage **never regenerates** — it remains open for the entire round and all subsequent rounds of the half |
| **Callout** | The opening is called "B Door" or "B Wall open" from the B side |
| **Tactical note** | Breaking it creates a third angle into B site; defenders must reposition to cover it; the noise alone reveals attacker position and intent |

---

## Silent Drop — Mechanics Reference

| Feature | Detail |
|---|---|
| **Location** | From A Top (Heaven) down to A Drop |
| **Mechanic** | Holding **Forward** movement then tapping **Backward** (or Shift-walk) immediately before landing eliminates footstep noise |
| **Failed execution** | Incorrect technique produces audible footstep sounds, alerting defenders |
| **Tactical use** | Allows flanks or repositions from A Top to the area near Defender Spawn / Waterfall without audio tell |

## Strategic Notes for Relay Corpus

- **Rotating doors are the map's information heartbeat** — every door activation is audible and visible; call it INSTANTLY ("Door A opening", "Door C"). Defenders riding a rotating door silently reposition; attackers can fake with a door activation.
- **Destructible wall (B Wall) is a one-time commitment** — breaking it is loud, permanent, and reveals attacker intent; call "Wall broken" the second it goes down; defenders must react immediately.
- **Silent drop is a premium flank tool** — a defender silently dropping from A Heaven can catch retaking attackers completely off-guard; always check A Drop on post-plant.
- **B site is the execution site** — smallest and most direct; with B Pillars smoked + A Link pressure (or wall broken), it becomes a clean three-man rush; strongest for economic advantage rounds.
- **A site is the complex site** — two entry angles (A Root / A Main + A Tree door), elevated heaven, a silent drop, and a site-to-site connector make A the hardest to defend alone; requires at least two defenders.
- **C Mound control is the C-side equivalent of Garage on Haven** — whoever takes Mound first controls the C Door timing and the B-side pressure; losing Mound early puts defenders in a reactive position.
- **C Gravel / C Hall audio** — the gravel audio in C Gravel gives the defender free information on flanks; listen for footsteps before committing to push C Hall.
- **Three sites = always one short** — with only five defenders across three sites, attackers who establish map control on two lanes simultaneously (A Main + C Mound, or B Pillars + A Root) force the under-manned site to buckle.
- **Ult orbs at mid and both sites** — three ult orbs spread across the map; contesting mid orb (B Upper area) is a common early-round priority fight.
- **Viper wall / Harbor wall for A Main** — A Main's length makes it an ideal smoke-wall axis; cutting it mid-corridor gives attackers safe approach to A Link and B site.
